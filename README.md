# Data-Co-Order-status-Risk-monitor-dashboard (Interactive dashboard using Power BI)
## Project Objective
DataCo wants to understand the business impact of production delays caused by late supplier deliveries.

## Dartaset used
-<a href="https://www.kaggle.com/datasets/shashwatwork/dataco-smart-supply-chain-for-big-data-analysis">
## KPI/Questions
- How can we improve the on-time order rate to increase customer satisfaction?
- What measures can be taken to reduce the delayed orders percentage, particularly in the "Standard Class" shipping mode?
- What impact do product category discounts have on sales performance and profit margins?
- Which countries have the highest number of delayed orders, and how can we address the root causes?
- How does the delayed order trend by year (2015-2018) affect overall business performance?
